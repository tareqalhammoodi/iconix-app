# Iconix

Iconix is a focused web tool for generating complete app icon sets and tracing PNGs into clean SVG vectors from a single upload. After trying other tools that were ad-heavy, paid, or unreliable, I decided to build my own. Feel free to use it yourself [here](https://iconix-app.vercel.app/).

## Features

- Generate iOS, Android, web, and Flutter icon packs from PNG or SVG uploads.
- Export zipped icon folders following common platform conventions.
- Convert PNG images to SVG with adjustable tracing controls and instant previews.
